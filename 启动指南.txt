# 智能旅行助手启动指南

## 环境要求
- Node.js >= 18
- Python >= 3.12
- 高德地图 API Key
- OpenAI API Key (或其他LLM服务)

## 后端启动

### 1. 安装依赖
```bash
cd backend
pip install -r requirements.txt
```

### 2. 配置API密钥
编辑 `backend/.env` 文件，填写你的API密钥：
```
# 高德地图API Key (必须)
AMAP_API_KEY=你的高德地图API密钥

# LLM服务配置
LLM_API_KEY=你的OpenAI API密钥
LLM_BASE_URL=https://api.openai.com/v1
LLM_MODEL_ID=gpt-4o-mini

# 服务器配置 (可选)
HOST=0.0.0.0
PORT=8000
DEBUG=true
```

### 3. 启动后端服务
```bash
# 使用默认端口8000
python3 -m uvicorn app.main:app --reload --port 8000

# 如果8000端口被占用，使用其他端口
# python3 -m uvicorn app.main:app --reload --port 8001
```

## 前端启动

### 1. 安装依赖
```bash
cd frontend
npm install
```

### 2. 启动前端服务
```bash
# 使用默认端口5173
npm run dev

# 如果需要指定端口
# npm run dev -- --port 5174
```

## 访问地址
- 前端：http://localhost:5173 (如果端口被占用，会自动使用其他端口，如5174)
- 后端API文档：http://localhost:8000/docs

## 一键启动脚本
项目根目录提供了 `start.sh` 脚本，可以一键启动前后端：

```bash
# 赋予执行权限
chmod +x start.sh

# 运行一键启动脚本
./start.sh
```

## 端口号被占用的解决方案

### 后端端口被占用
```bash
# 查看端口占用情况
lsof -i :8000

# 终止占用端口的进程
kill -9 进程ID

# 或使用其他端口
python3 -m uvicorn app.main:app --reload --port 8001
```

### 前端端口被占用
```bash
# 方法1：使用npm命令指定端口
npm run dev -- --port 5174

# 方法2：修改vite.config.ts文件
// 在vite.config.ts中添加
server: {
  port: 5174
}
```

## 常见问题解决

### 1. uvicorn 命令未找到
```bash
# 使用python -m uvicorn
python3 -m uvicorn app.main:app --reload --port 8000
```

### 2. API密钥错误
```bash
# 检查.env文件中的API密钥配置
# 确保使用正确的环境变量名称：LLM_API_KEY, LLM_BASE_URL, LLM_MODEL_ID
```

### 3. 前端依赖安装失败
```bash
# 清理npm缓存
npm cache clean --force

# 重新安装依赖
npm install
```
